# Debit Repayment Planner

A full-stack application for managing debt repayment with Spring Boot backend and React frontend.

## Project Structure
```
fef87ee5-1cdd-4da0-af4b-26eb1398050f-0f776816-8c46-4b78-ad10-aad695291552/
├── springapp/          # Spring Boot backend
└── reactapp/           # React frontend
```

## Quick Start

### Backend (Spring Boot)
```bash
cd fef87ee5-1cdd-4da0-af4b-26eb1398050f-0f776816-8c46-4b78-ad10-aad695291552/springapp
./mvnw spring-boot:run
```

### Frontend (React)
```bash
cd fef87ee5-1cdd-4da0-af4b-26eb1398050f-0f776816-8c46-4b78-ad10-aad695291552/reactapp
npm install
npm start
```
